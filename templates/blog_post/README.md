# Blog post template

Copy `blog_posts/_template/` to `blog_posts/my_slug/`.

**Required:** include the site header from `templates/snippets/blog_site_header.html` so navigation persists when reading a post.

For rich posts with audio/video, see `blog_posts/live_music_diffusion_models/` as the reference implementation. **Do not commit `.mp4` files** — upload videos to YouTube and embed them (see `CONTRIBUTING.md`).

Required meta: `type`, `date`, `title`, `subtitle`, `excerpt`.
